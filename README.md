# Gym-Flappy-Bird

Gym-Flappy-Bird is a flappy-bird game bot trained using Q-learning.

## Installation

Simply clone this project.


## Usage

```bash
# run a demo
python src/demo.py

# training using epsilon-greedy
python src/train_eg.py

# training using UCB
python src/train_ucb.py
```
